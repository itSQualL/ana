# README

## RUN WITH DOCKER

Install docker
Install docker-compose

Run: `docker-compose build web`

Run: `docker-compose run web bundle exec rake db:create`

Run: `docker-compose run web bundle exec rake db:migrate`

Run: `docker-compose up web`
